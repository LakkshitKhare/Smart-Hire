# SmartHire — Smart Interview Assessment & Hiring Platform

SmartHire is a full-stack enterprise hiring platform built using **Spring Boot**, **Java 21**, and **Angular**.  
The platform helps companies streamline recruitment by managing candidates, assessments, interviews, and hiring workflows in one place.

---

# 🚀 Features

## Authentication & Authorization
- Secure JWT Authentication
- Role-Based Access Control
- BCrypt Password Hashing
- Login & Registration System

## User Roles
- Candidate
- Recruiter
- Admin
- Interviewer

## Candidate Features
- Registration & Login
- Profile Management
- Resume Upload
- Job Applications

## Recruiter Features
- Create Job Openings
- Manage Candidates
- Track Hiring Pipeline
- Schedule Interviews

## Assessment Features
- Online Coding Tests
- MCQ Assessments
- Timer-Based Exams
- Auto Evaluation

## Future Features
- AI Resume Screening
- Video Interviews
- Real-Time Notifications
- Analytics Dashboard
